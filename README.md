# Winds Adventure - Martial Arts Combat Game

A 3D martial arts combat arena game built with React Native (Expo) and NestJS.

## Features

- 3D Combat Arena with Three.js
- Wave-Based Combat System
- Character Progression and Leveling
- Inventory System with 30+ Items
- Quest System with 15 Quests
- Leaderboard

## Tech Stack

### Frontend
- React Native (Expo)
- TypeScript
- Three.js for 3D graphics
- React Navigation
- React Native Paper

### Backend
- NestJS
- TypeScript
- PostgreSQL
- Prisma ORM
- JWT Authentication

## Game Mechanics

### Combat System
- Light Attack (10 SP): Quick damage
- Heavy Attack (20 SP): More damage
- Special Attack (30 SP): Massive damage
- Dodge (15 SP): Evade attacks

### Enemy Types
- Basic (100 HP)
- Elite (200 HP)
- Boss (500 HP)
